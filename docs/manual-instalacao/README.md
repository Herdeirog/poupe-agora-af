# Manual de Instalacao - Poupe Agora 4.0

Guia completo para instalar o sistema do zero. **Siga as etapas na ordem exata.**

---

| # | Etapa | Descricao |
|---|-------|-----------|
| 01 | [Criar Projeto Supabase](01-criar-projeto-supabase.md) | Criar conta e projeto no Supabase |

| 03 | [Criar Usuario Admin](03-criar-usuario-admin.md) | Criar o primeiro usuario administrador |
| 04 | [Configurar Secrets](04-configurar-secrets.md) | Adicionar chaves nas Edge Functions |

## Requisitos

Antes de comecar, voce precisa ter:

- **Node.js** 18+ instalado (recomendado 20+)
- **Git** instalado
- **npm** como gerenciador de pacotes
- Uma conta no **Supabase** (gratuita funciona)
- Uma conta no **Vercel** (para producao)
- (Opcional) Conta na **OpenAI** para agentes de IA
- (Opcional) **Evolution API** para integracao WhatsApp
